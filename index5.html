مقدمه

ngrok یک ابزار کراس پلتفرم است که به توسعه‌دهندگان اجازه می‌دهد تا سرویس‌های لوکال خود را در اینترنت در معرض دید قرار دهند. به عبارت دیگر، ngrok یک تونل امن از طریق اینترنت به ماشین توسعه شما ایجاد می‌کند. و برای هک استفاده میشود.

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

با این دستور نصبش میکنیم

git clone https://github.com/Yisus7u7/termux-ngrok

با این دستور وارد پوشش میشیم

cd termux-ngrok

و با این دستور ابزار رو اجرا میکنیم

bash install.sh

رو میخواد ngrok اینتر رو ک بزنید ازتون توکن حساب

خب وارد این سایت بشید و با یه ایمیل ثبت نام کنید

توجه داشته باشید شاید نیاز باشه با فیلتر شکن وارد بشید

برای ورود به سایت کلیک کنید

وارد حساب کاربری خودتون بشید و با بالا سمت چپ روی دکمه سه خط بزنید و از داخل گزینه ها گزینه توکن رو انتخاب کنید اینجا توکن شما رو میده و یکم پایینتر یه کادر هست که یه گزینه داره وقتی اونو بنزید توکن شما رو با یه کد اولش میده که باید اونو کپی کنید و داخل ترموکس بزنید.

اینجا وقتی اینتر رو بزنید نصب میشه و برای تست این کد رو بزنید

ngrok http 3333

بعدش وارد یه صفحه دیگه میشه و یه لینک بهتون میده
