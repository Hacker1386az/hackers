مقدمه

بمباران پیامکی (SMS Bomber) یک نوع حمله سایبری است که در آن تعداد زیادی پیامک به طور مکرر به یک شماره تلفن ارسال می‌شود. این عمل با هدف ایجاد مزاحمت، مختل کردن سرویس تلفن همراه قربانی و یا پوشاندن پیام‌های مهم انجام می‌شود. بمباران پیامکی می‌تواند از طریق نرم‌افزارها یا وب‌سایت‌های خاصی که این سرویس را ارائه می‌دهند، انجام شود. این حملات معمولاً غیرقانونی هستند و می‌توانند منجر به پیگرد قانونی شوند. هدف اصلی از بمباران پیامکی ایجاد اختلال و مزاحمت برای فرد یا سازمانی است که مورد حمله قرار گرفته است.

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

با این دستور نصبش میکنیم

git clone https://github.com/M-logique/Iran-Bomber/

با این دستور وارد پوشش میشیم

cd Iran-Bomber

با این دستور ابزار های مورد نیاز رو نصب میکنیم

pip install -r requirements.txt

و با این دستور ابزار رو اجرا میکنیم

python main.py

اینتر رو ک بزنید دوتا گزینه میاد

۱.کال بمبر

۲.اس ام اس بمبر

هرکدوم رو بزنید بعد شماره میخواد

شماره طرف رو بدون صفر وارد میکنید و اینتر رو میزنید و بعد تعداد رو میخواد مثلا میزنید ۱۰۰ بعد اینتر و بومم 🦂
