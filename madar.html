مقدمه

این کار با دو روش انجام میشه اول اینکه از طریق ایپی دوربین اونو کرک میکنیم . و روش دوم اینجوریه که یه مقدار دوربین های مشخص شده رو میتونید ببینید

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

ابزار اول

git clone https://github.com/threat9/routersploit

cd routersploit

python3 -m pip install -r requirements.txt

python3 py 4

با اجرای دستورات فوق کار روتر اسپلویت آغاز می شود. بعد از آن برای شروع هک دوربین باید عبارت زیر را تایپ کنید:

use scanners/cameras/camera_scan

show options

set target xx.xx.xx.xx

به جای xx.xx.xx.xx آدرس IP دوربینی است که قرار است هک شود قرار می گیرد. در آخر دستور RUN را تایپ کنید.

روش دوم

با این دستور نصبش میکنیم

git clone https://github.com/M-logique/Iran-Bomber/

با این دستور وارد پوشش میشیم

cd Iran-Bomber

با این دستور ابزار های مورد نیاز رو نصب میکنیم

pip install -r requirements.txt

و با این دستور ابزار رو اجرا میکنیم

python main.py

اسم همه کشور ها هست و کنارش یه اسم مخفف زده

اسم مخفف هر کشوری که میخوایم رو میزنیم و اینتر

لیست ایپی های دروبین ها رو بهمون میده و میتونید تو گوگل سرچ کنید و ببینید

هست هم پسورد و هم یوزر admin بعضی هاشون رمز دارن و رمز بعضی از اونها
