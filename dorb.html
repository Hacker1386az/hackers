مقدمه

هک دوربین گوشی اینجوریه که این ابزار یه دسترسی دوربین از تارگت میگیره و تند تند از دوربینش برامون توی ترموکس عکس میفرسته . خب بریم برای اموزش.

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

با این دستور نصبش میکنیم

git clone https://github.com/mrmahdi8800/cam_test.git

با این دستور وارد پوشش میشیم

cd cam_test

و با این دستور ابزار رو اجرا میکنیم

bash cam.sh

اینتر رو ک بزنید دوتا گزینه میاد

گزینه ۲ رو بزنید و اینتر

یه صفحه جدید باز کنید

انگشتتون رو بزارید سمت چپ صفحه گوشی توی ترموکس و بکشید به سمت راست یه صفحه باز میشه که دوتا گزینه داره و باید بزنید روی گزینه سمت راست و یه صفحه جدید باز میشه

خب حالا تو صفحه جدید این کد رو بزنید و بعد اینتر . توجه داشته باشید باید قبل از این کار حتما آموزش راه اندازی و کار با نگروک رو دیده باشید .

ngrok http 3333

حالا لینک رو که بهتون میده رو کپی کنید و به تارگت بدید

الان باید برگردید به صفحه قبلی

حالا وقتی تارگت لینگ رو باز کنه ای پی طرف برای شما میاد و وقتی دسترسی بده براتون عکس ها میاد

برای دیدن عکس ها این کار ها رو انجام بدید

رو توی کیبوردتون بزنید تا متوقف بشه c خب اول باید دکمه ولوم پایین گوشیتون رو نگه دارید و همزمان دکمه

حالا این کد رو بزنید تا ذخیره بشه

mv *png /sdcard
